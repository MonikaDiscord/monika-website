Monika's commands

  • Commands marked with patron are only available for patrons. You can become a patron at patreon.com/monikabot..
  • Commands marked with voter are only available for voters. You can upvote me at discordbots.org/bot/monika/vote.
  • My default prefix is $!.

Here are my commands

Name Description Usage
General
help Tells you Monika's commands. help
info Displays information about Monika. info
server Provides an invite to Monika's server. server

Moderation

Administration

resetprefix manage server Resets your server's prefix. resetprefix
filter manage messages Shows a list of filtering commands (unless you provide a subcommand). filter {subcommand}
Music

skip manage messages Skips the song currently playing. skip
stop manage messages Clears and stops the queue. stop
now Displays what's currently playing. now
queue Displays the current queue. queue
disconnect manage messages Makes Monika disconnect from your VC. disconnect
Fun
